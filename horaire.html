Cours Préférés

Conception Web

Description :Ce cours est une introduction au développement d’applications Web non transactionnelles en utilisant les langages HTML et CSS.

Ce que j'ai le plus apprécié : J'ai trouver les explications simples et efficace pour la compréhension.

Ce que j'ai moins apprécié :Il y a que un seul gros cours sur une semaine ce qui est dommage.

Introduction à la Programmation

Description : Ce cours est une introduction aux méthodes généralement utilisées en programmation et en algorithmie pour développer des applications simples. Le langage de programmation utilisé est le C#.

Ce que j'ai le plus apprécié : J'ai beaucoup aimé résoudre des problèmes et apprendre pleins de nouvelles choses.

Ce que j'ai moins apprécié : Les examens théoriques sont assez difficile.
